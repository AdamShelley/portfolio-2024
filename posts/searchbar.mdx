Inspired by the likes sonner and shadcn, I wanted to create a searchbar component that is minimalist and nice to use.


There are many props to feed into the SearchBar, but here are a couple of demos below to get the general idea.
It's built with React, Tailwind, and **Motion** for the animations. 

This component forced me to think about how the user might need to interact with a Search bar, especially if the user is using 
an accessibility tool like a screen reader.
